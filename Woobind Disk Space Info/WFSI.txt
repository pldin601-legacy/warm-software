Woobind Disk Space Info
-----------------------

Версия 2.0


О программе
-----------

Программа Woobind Disk Space Info предназначена для отображения на рабочем столе 
количества свободного места на выбранных дисках в реальном времени.

Основные возможности
--------------------

+ Отображение текста прямо на картинке рабочего стола;
+ Отображение только при наведенном курсоре мышки;
+ Поддержка "прозрачного режима" при котором клики мышки проходят сквозь окно программы;
+ Быстрая установка;
+ Поддержка шаблонов для вывода информации;
+ Кроме текста также можно разместить индикатор.
